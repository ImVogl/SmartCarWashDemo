Для сборки проекта необходимо усстановить .NET 5.0 SDK и из текущей папки нужно выполнить команду dotnet build . -c release
После этого запустить .\SmartCarWashDemo\bin\Release\net5.0\SmartCarWashDemo.exe
В консоли появится зпись Now listening on: http://localhost:5000 или аналогичная ей.
Далее в браузере нужно перейти по следующему адресу http://localhost:5000/swagger/index.html или с тем портом, который был указан при запуске.

В базе есть два пользователя с id 1 и 2
Три точки продажи 1, 2, 3
В сведениях о точки продажи содержатся идентификаторы товаров.
Есть два акта продажи с идентификаторами 1 и 2
